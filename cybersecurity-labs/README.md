## Cybersecurity Lab Environment

This repository will document the design and setup of a hands-on cybersecurity lab.

### Planned Lab Components
- SIEM platform
- Windows endpoint (event logs, Sysmon)
- Linux server (auditd)
- Simulated attack traffic for detection testing

### Goal
To understand log collection, correlation, alerting, and investigation from a SOC analyst perspective.
